TOEFL 自学指北

发表于 2018-06-26 | 分类于 出国 |

0. 前言

在今年一月杀鸡成功以后，就踏上了自学托福的路程。从 3 月初开始备考到 5 月 26 号考试，经历了大概三个月的时间。最后成绩是 R29 + L30 + S23 + W24 = 106，虽然比不上票圈很多大佬的 110+，但是也算是够用的了成绩。于是就记录一下备考的一些经验/经历，以期能给自己一个回忆，且给将要考试的同学一些建议。

既然托福分成四个部分考，那我就从它们每个部分分开来讲啦。

1. 阅读

单词

我是先考的 GRE，所以在转到托福的阅读时，会发现这些单词都好熟悉。所以做好阅读的第一步，就还是打好单词的基础。新东方红宝书至少要掌握90%。如果你的基础单词比较 OK，那么在做词义题的时候基本就可以不看文章秒答。这种题每篇文章都有 2～3 题，能直接做出来还是感觉蛮爽的。另外由于托福文章的专业性都比较强，所以专有名词的储备也要有，这点可以参考《新东方 词以类记：TOEFL iBT词汇》。不过这种专业词汇不用特别着急记，可以在后面的做题中逐渐积累。

逻辑

托福文章的逻辑都比较简单，不像 GRE 要绕好几个弯。所以主要是要分清段间关系，这样在做总结题的时候知道什么句子大概在哪里找。而且托福文章很棒的一点就是出题的顺序和文章顺序对应，所以可以边看文章边做题。

托福的阅读总体不是很难，有一大半的题都是单词、找事实、找反面事实、找指代，只要认真仔细就能做对。比较难的是插句子、转述句子、总结题。

所谓插入句子题，让你把一个句子插到一段内四个地方中的一个，使意思连贯。这时候就要强烈关注段内和待插句子中的连接词与代词。如果你能很清楚的知道句内 this/that/it 指代的是什么，也就知道了他该接在谁后面。掌握这个技巧会很有用。

转述句子，就是让你选出一个和选中句子意思最相近的选项。这时候有几个原则。第一，太短的一般都不会选，因为太短意味着失去了很多句子的重要信息。当然也不绝对，只是大部分情况。第二，很关键的动作、名词不能丢掉，不然句子可能会少掉一个组成部分。第三，在选好答案以后，把你的答案替换掉原文的句子，然后重读那一段，看看上下文逻辑能否通顺。

最后的总结题，一般是六选三。一般都能很轻松地去掉一到两个一看就错的答案（比如不符合常识，或者在文中没出现过）。接下来，只要仔细分析选项的主语是不是文章的主体，以及把它去掉能不能影响文章的结构，基本就可以确定所有答案。所以最后一题靠的是仔细，和对文章整体主旨的把握。

2. 听力

托福听力分两种，对话和讲座。我觉得略难的是讲座，毕竟他的信息量大。下面就分别讲解一下。

注：以下做笔记方法仅供参考，最好还是以自己习惯的笔记方式进行。

对话

对话一般是学生和一个教职工或者学生和学生之间的对话。首先，要迅速分清说话的双方的身份，并在草稿纸上写下两栏，比如：Stu | Prof。然后随着对话的进行，在每个人下面写下他们的关键字，如动作、原因，然后可以在两栏之间画箭头，表示支持、反对、帮助等态度或动作。这样最后做题的时候，至少大方向上不会选错。另外，有的细节的东西能记也要记，因为你不知道他会问什么，很可能就会问“教授喜欢的画的名字是什么？”这种细枝末节的问题。所以如果能记下一个简写也是好的。

讲座

讲座的话，其实我觉得有点看运气，如果他的内容你本来就熟悉（比如讲太阳系的，大家都比较清楚），那可能听的时候讲游刃有余一点。如果内容不熟悉，又碰到了一个澳大利亚/英国口音的老师（只是我们不太习惯），那就要多注意一下了。关于讲座的笔记，我觉得还是以关键字+图示为主。比如有很多流程类的，就要在关键字间多画箭头，比如：A ⬇️ ➡️ B ⬆️，表示 A 减少导致 B 上升。总之画图应该是比写字省时间的。另外对一些有明确分类的，一定要做笔记，因为很可能最后的答题就是让你归类这些东西。当时 TPO 有一题就没记下来这些所以最后大题一分没得。

笔记

关于如何做笔记众说纷纭，有的说能记多少记多少，有的说少记点多听文章。而我觉得，依据你个人的实力（如听力的接受程度、写字的速度、速记的翻译速度）稍作调整。最好的情况是：文章的整体思路明确，少许细节做笔记，整个过程中 50%～70% 时间是在听，其余时间边听边记。毕竟，你写字的速度肯定跟不上他说话的速度。这个比例可以在大量练习的过程中自行调整。

3. 口语

口语应该是中国学生做头疼的部分，包括我。我当时自己练习的时候，一开始都羞于突然开始说一段 60 秒的英文，这很可能让你身边的人觉得你受了什么刺激😄。不过克服心理障碍以后，你就当是在跟电脑推心置腹就可以了。

不过我的口语也没有考很高，可能是跟 Q3、Q4 没完全说完论点有关系，默默可惜一下。

笔记

口语也是有很多阅读和听力的部分的，所以记笔记的方法参照上述。在有些题目上（比如 Q3），只要记下阅读材料中最关键的问题就行，而且能抄原文就抄原文，到时候开头直接说一部分原文并不是会特别影响你的分数，反而会增加你的流利程度和自信。

练习

我没经过培训班的训练，所以练习口语用的只是自己的野方法。打开手机的录音 App，然后每做一题，都录下来。如果做的题有参考答案，听一遍参考答案，然后自己再重新说一遍。最后把自己的两次答案都听一遍，寻找之间哪里不同，告诉自己下一次答题时候在什么地方要主动特别注意。渐渐地你就会形成一个习惯，很自然地改掉一些问题。

模板

口语每道题基本都是有模板的，粒度有大有小，从一整个大的框架到具体的每句话怎么说，各个材料给出的不尽相同。而我觉得总的结构应该是这样：态度 + 主题1 + 例子1 + 主题2 + 例子2。当然如果你时间不够，只说一个例子也行，但是一定要有理有据、支持主题。只要符合这个套路，一般都没问题。我觉得可以多看一些模板，然后自己多记几种句式，到时候换着用，你觉得哪个最顺口就说哪个。

4. 写作

我的写作依然是考场处女作，结果是 Fair + Good。看来是第一篇综合写作的锅，因为在听听力的时候有一个反驳的原因没有记得很清楚。所以啊，不管口语还是写作，关键的还是听力！如果听力不过关，写作时候你可能都不知道教授为什么反对给出的材料，那就更无从下笔了。

独立写作我直接用的 GRE 的套路，后来发现还挺好用。唯一的差别就是你要多注意自己的语言，千万不要有低级语法错误（因为会很扣分），然后能用高级的句式就用，什么从句倒装强调都可以用上。另外要多积累一些替换词，比如 choice 改成 alternative 这种，让你的文章看起来丰富一点。最后把你的文章通读几遍，当你觉得它看起来还挺像一篇 native speaker 的文章的时候基本就可以了。

5. 心态

这是我第一次考托福，还是自学的，还要当大家的面说口语（我就是脸皮太薄），刚进考场还是有点紧张的。不过事实证明不要给自己太大的心理压力，考试时候大家都是专注于自己的没人会管你怎么样。另外特别感谢我们南大的 C 考场（大家一定要选这个！！！），每个人坐在一个办公室一样的隔间里，受旁边人影响不大。不像 A 和 B，每个人用一个弯着的泡沫纸板隔开，简直安全感为零。

考试时候你就当是平时练习一样做题就行了，一开始做阅读时候可能有点紧张，但是没关系，做了前面几道简单的题你就会逐渐进入状态。说口语的时候声音要大，让别人感觉到你的自信，然后自己的信心也就上来了。甚至做些习惯的手势辅助自己的都没关系，反正没人在看你。写作时候也可以用力敲键盘，感觉自己下笔如有神，尤其是空格键可以大拇指用力敲下去。哈哈最后一条这是说笑，不过用各种方式让自己自信是很重要的，你都做那么多 TPO 了还差这么一套真题吗哈哈。

6. 材料

书

我用的是《新东方 新托福考试专项进阶—高级口语+听力+写作+阅读》，听说是韩国人编的，事实证明也是偏简单，可以入门和熟悉考试用。

TPO

我是在考满分上练的托福 TPO 1～20。然而，直到考试前一天我才知道前 20 套是比较简单的，当时心态就崩了。所以大家在做 TPO 的时候一定要前面的为辅，后面的为主，后面的比较贴近当今考试难度。

对了，在考满分上做模考他会给你个估分（我练的时候只有阅读听力可以评分，现在好像口语也行了），我做阅读一般是 25 ～ 30，听力是 22 ～ 30，波动比较大。不过放心，最终考试的分数肯定会比模考高。如果你模考全都满分了，培训机构还挣什么钱呢😁。

7. 广告

需要杀 G 的同学可以参考我的另一篇文章啦：GRE 自学指北

最后祝大家早日和托福顺利分手！

GRE 自学指北

发表于 2018-03-14 | 分类于 出国 |

前言

今年1月20号考了人生中第一次可能也是最后一次 GRE，总分是 V 159 + Q 170 + AW 4.0，还算凑合吧。当时也纠结了一下要不要上个培训班，后来还是一狠心没去（想省钱，笑）。今天写篇文章来记录一下当时是怎么备考的，分享一些小小的经验，以期对即将备考/考试的同学有所帮助。

准备

首先你要对 GRE 要考什么和自己的能力有些了解，然后要很清楚你相较于从容考试的能力还差在什么地方，那么这些地方就是你要认真补充和学习的。所以此处对 GRE 的考试规则之类就不赘述，而主要谈谈我感觉中，GRE 整体在考的两块东西：词汇和逻辑。

词汇

首先说词汇。GRE 的单词难也是大家有所耳闻的，所谓的“大3000”也是让人听了退避三尺。据说，GRE 要求的单词量在12000左右。这其中你可能真的会碰到一些“除了考试，这辈子再也遇不到”的单词。所以，为了打下单词关，其实还是需要一点积累和努力的。

如果你之前考过托福、背过了托福单词，或者单词量在7-8000左右，那么我觉得你可以直接买本“大3000”开始背；如果觉得自己还稍弱一些，那么可以先找本托福单词入手。因为那本大3000已经默认你掌握了大部分托福单词，所以这两者结合起来才能保证你考试时候从容不迫。

另外，有人问是不是要把大3000所有单词都会背？我觉得这也要分情况讨论。如果你还有一年才去考，那你大可放心地慢慢把这些全背出来。如果你还有三个月，那么可能你只有一个月准备单词，这时候就要快速的把大3000里你已经认识的单词划掉（对，用黑笔划掉，看不见的那种），然后主攻剩下的单词。直到你把整本书背出来70%-80%，那就可以开始做题了。因为其实 GRE 题目里常考的单词也有一个范围，多做题以后你就发现，他爱考的就是那么一些词，那么你把这些词多背背熟就能较高地提高背单词的“性价比”。这个阶段就是要开始以题代背，不要过多地硬背单词。

而且，要多注意总结其中的同义词和反义词，而且要至少知道这个单词是褒义还是贬义，这对做题会大有裨益。

逻辑

接下来就全是逻辑了。其实阅读、数学、作文，每个部分都是在考逻辑。

阅读

词汇题

刚说完词汇，那么看看词汇跟逻辑有啥关系。

GRE 词汇题分两种：单空题和多空题（两空或者三空）。单空题会让你从六个选项里选两个填到句子里都合适的单词，多空题让你在每个空里从一组词中选择一个，使句子意思正确。

对于单空题：首先知道哪两个单词意思相近很重要，总不能选一个 good 一个 bad 进去，那肯定不会对。所以，第一步就是找到两个意思相近、至少倾向相同（都是褒义/贬义）的词，这时候基本就只剩下两组（6个中的4个）了。这时候再看看句子中的关键字，比如 however、but、otherwise 之类，他们会提示你要填的词和另一个分句的词中的情感是相同还是相反。比如，如果 however 后面是一个 happy 的词，你前面填负面的那组词就肯定对了。有时候你甚至都不用知道选项具体是啥意思，只要知道他是个“坏词”就好。

多空题类似，不过只是因为他句子比较长（甚至一道题等于一个自然段），所以要缜密分析其中的逻辑。这时第一不应该是标注关键字，然后把句子分段，看每一段的情感之间是相同还是相反。再从选项中根据倾向填词，正确率会高很多。不过三空是有点难了，因为他经常会用从句误导你，你以为 B 跟 C 有关系，其实 A 跟 C 才有关系。这时候就要好好分析清楚。

阅读题

阅读又分为逻辑单题、短文章和长文章。而题目考的很多也都是句间关系和段间关系。那么这时候比较重要的就是先略读文章，把文章的结构分析清楚，而不拘泥于细节。要是真抠细节题，你文章都看过一遍了再去找也来得及。对于逻辑关系很强的段落，比如时间关系、因果关系，就要多加重点因为他们很爱考这个。一定要搞明白，是 A 造成了 B，B 造成了 C，C 之后才有 D 之类。实在不行你都可以用上一阶谓词逻辑，not A → B，B → not C 这样写出来会清楚很多。

另外有的题用常识都能判断，就不用看文章了。

文章里可能出现一堆专有名词（比如最爱考的 zooplankton），别怕，把它们全用首字母代替就行。只要不影响逻辑，你想换成小明也没问题。

做题我推荐“陈虎平36套”，讲解很详细。

数学

数学一看就是考逻辑啦。不过数学对中国学生一般都没啥问题，停留在初中水平，最难得可能就是把专有名词记记牢（比如菱形：rhombus 考 GRE 之前我是真不知道）。有一些常跳的坑，大家可以参照猴哥难题，多刷一刷找找感觉，168以上应该没问题。

作文

说来惭愧，我在考场才写了第一篇正式 GRE 作文。

作文有两篇：Argument 和 Issue。其中 Argument 比较简单。

Argument

这篇就是让你读一段文字，然后找出其中的逻辑漏洞。其实还蛮好找的，因为一共就那么几类：样本不足、不合理外推、错误类比、调查没根据……大概七八种。你就一一对照那个段落去看，直到找出3大类错误，然后每一类错误大概找1-2个攻击点，再一个开头一个结尾即可。开头结尾都有模板的，可以参照“GRE 作文大讲堂”。

Issue

这篇就有点议论文的意思，让你自己阐述观点。其实跟上面文章的结构差不多，也是开头+结尾，中间大概三到四段，其中2个支持（或反对）观点的，1个反对（或支持）观点的，以展示出你能全面辩证地看待问题。每一段一定要写中心句，后面跟点说理和例子。千万别举自己的例子，因为毕竟我们（应该）不是名人哈哈哈。

一些注意事项：

  • 作文字数一定要够，这很大程度决定你的分数。至少要写到答题页面右边的滚动条出现，最后再多出一段的样子。
  • GRE 不是语言考试是逻辑考试，不要想着用大词，我当时写的都有点口语化了，主要是要把逻辑写清楚就行。
  • GRE 的作文是有题库的！说不定你运气好考试前看了一篇第二天就考到了。所以在平时练的时候，看一篇就至少要把逻辑架构想一想，不要考试时候后悔啦。

杂项

最后说说心态吧。要知道，GRE 是自适应评分，如果你前面做的好，后面题就会变难，不过总分会高一个档次。所以如果你遇到难题，不要气馁，反而这可能是你做的好的表现。当时我做阅读最后一两篇时间都不太够加上紧张，所以就没几个对的，后来看我的分析，那几个题都是4-5级的题（GRE 的题难度1-5，5最高），如果再多对一两个可能分数会好看一些。所以千万不要紧张啊~

考试时候带个小矿泉水，然后一些高热量食物。因为考试时间还挺长的，不要到时候大脑供血不足。

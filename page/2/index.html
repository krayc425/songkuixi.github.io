GRE 自学指北

发表于 2018-03-14 | 分类于 出国 |

前言

今年1月20号考了人生中第一次可能也是最后一次 GRE，总分是 V 159 + Q 170 + AW 4.0，还算凑合吧。当时也纠结了一下要不要上个培训班，后来还是一狠心没去（想省钱，笑）。今天写篇文章来记录一下当时是怎么备考的，分享一些小小的经验，以期对即将备考/考试的同学有所帮助。

准备

首先你要对 GRE 要考什么和自己的能力有些了解，然后要很清楚你相较于从容考试的能力还差在什么地方，那么这些地方就是你要认真补充和学习的。所以此处对 GRE 的考试规则之类就不赘述，而主要谈谈我感觉中，GRE 整体在考的两块东西：词汇和逻辑。

词汇

首先说词汇。GRE 的单词难也是大家有所耳闻的，所谓的“大3000”也是让人听了退避三尺。据说，GRE 要求的单词量在12000左右。这其中你可能真的会碰到一些“除了考试，这辈子再也遇不到”的单词。所以，为了打下单词关，其实还是需要一点积累和努力的。

如果你之前考过托福、背过了托福单词，或者单词量在7-8000左右，那么我觉得你可以直接买本“大3000”开始背；如果觉得自己还稍弱一些，那么可以先找本托福单词入手。因为那本大3000已经默认你掌握了大部分托福单词，所以这两者结合起来才能保证你考试时候从容不迫。

另外，有人问是不是要把大3000所有单词都会背？我觉得这也要分情况讨论。如果你还有一年才去考，那你大可放心地慢慢把这些全背出来。如果你还有三个月，那么可能你只有一个月准备单词，这时候就要快速的把大3000里你已经认识的单词划掉（对，用黑笔划掉，看不见的那种），然后主攻剩下的单词。直到你把整本书背出来70%-80%，那就可以开始做题了。因为其实 GRE 题目里常考的单词也有一个范围，多做题以后你就发现，他爱考的就是那么一些词，那么你把这些词多背背熟就能较高地提高背单词的“性价比”。这个阶段就是要开始以题代背，不要过多地硬背单词。

而且，要多注意总结其中的同义词和反义词，而且要至少知道这个单词是褒义还是贬义，这对做题会大有裨益。

逻辑

接下来就全是逻辑了。其实阅读、数学、作文，每个部分都是在考逻辑。

阅读

词汇题

刚说完词汇，那么看看词汇跟逻辑有啥关系。

GRE 词汇题分两种：单空题和多空题（两空或者三空）。单空题会让你从六个选项里选两个填到句子里都合适的单词，多空题让你在每个空里从一组词中选择一个，使句子意思正确。

对于单空题：首先知道哪两个单词意思相近很重要，总不能选一个 good 一个 bad 进去，那肯定不会对。所以，第一步就是找到两个意思相近、至少倾向相同（都是褒义/贬义）的词，这时候基本就只剩下两组（6个中的4个）了。这时候再看看句子中的关键字，比如 however、but、otherwise 之类，他们会提示你要填的词和另一个分句的词中的情感是相同还是相反。比如，如果 however 后面是一个 happy 的词，你前面填负面的那组词就肯定对了。有时候你甚至都不用知道选项具体是啥意思，只要知道他是个“坏词”就好。

多空题类似，不过只是因为他句子比较长（甚至一道题等于一个自然段），所以要缜密分析其中的逻辑。这时第一不应该是标注关键字，然后把句子分段，看每一段的情感之间是相同还是相反。再从选项中根据倾向填词，正确率会高很多。不过三空是有点难了，因为他经常会用从句误导你，你以为 B 跟 C 有关系，其实 A 跟 C 才有关系。这时候就要好好分析清楚。

阅读题

阅读又分为逻辑单题、短文章和长文章。而题目考的很多也都是句间关系和段间关系。那么这时候比较重要的就是先略读文章，把文章的结构分析清楚，而不拘泥于细节。要是真抠细节题，你文章都看过一遍了再去找也来得及。对于逻辑关系很强的段落，比如时间关系、因果关系，就要多加重点因为他们很爱考这个。一定要搞明白，是 A 造成了 B，B 造成了 C，C 之后才有 D 之类。实在不行你都可以用上一阶谓词逻辑，not A → B，B → not C 这样写出来会清楚很多。

另外有的题用常识都能判断，就不用看文章了。

文章里可能出现一堆专有名词（比如最爱考的 zooplankton），别怕，把它们全用首字母代替就行。只要不影响逻辑，你想换成小明也没问题。

做题我推荐“陈虎平36套”，讲解很详细。

数学

数学一看就是考逻辑啦。不过数学对中国学生一般都没啥问题，停留在初中水平，最难得可能就是把专有名词记记牢（比如菱形：rhombus 考 GRE 之前我是真不知道）。有一些常跳的坑，大家可以参照猴哥难题，多刷一刷找找感觉，168以上应该没问题。

作文

说来惭愧，我在考场才写了第一篇正式 GRE 作文。

作文有两篇：Argument 和 Issue。其中 Argument 比较简单。

Argument

这篇就是让你读一段文字，然后找出其中的逻辑漏洞。其实还蛮好找的，因为一共就那么几类：样本不足、不合理外推、错误类比、调查没根据……大概七八种。你就一一对照那个段落去看，直到找出3大类错误，然后每一类错误大概找1-2个攻击点，再一个开头一个结尾即可。开头结尾都有模板的，可以参照“GRE 作文大讲堂”。

Issue

这篇就有点议论文的意思，让你自己阐述观点。其实跟上面文章的结构差不多，也是开头+结尾，中间大概三到四段，其中2个支持（或反对）观点的，1个反对（或支持）观点的，以展示出你能全面辩证地看待问题。每一段一定要写中心句，后面跟点说理和例子。千万别举自己的例子，因为毕竟我们（应该）不是名人哈哈哈。

一些注意事项：

  • 作文字数一定要够，这很大程度决定你的分数。至少要写到答题页面右边的滚动条出现，最后再多出一段的样子。
  • GRE 不是语言考试是逻辑考试，不要想着用大词，我当时写的都有点口语化了，主要是要把逻辑写清楚就行。
  • GRE 的作文是有题库的！说不定你运气好考试前看了一篇第二天就考到了。所以在平时练的时候，看一篇就至少要把逻辑架构想一想，不要考试时候后悔啦。

杂项

最后说说心态吧。要知道，GRE 是自适应评分，如果你前面做的好，后面题就会变难，不过总分会高一个档次。所以如果你遇到难题，不要气馁，反而这可能是你做的好的表现。当时我做阅读最后一两篇时间都不太够加上紧张，所以就没几个对的，后来看我的分析，那几个题都是4-5级的题（GRE 的题难度1-5，5最高），如果再多对一两个可能分数会好看一些。所以千万不要紧张啊~

考试时候带个小矿泉水，然后一些高热量食物。因为考试时间还挺长的，不要到时候大脑供血不足。

推荐
